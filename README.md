# Website Prototype: Small Consulting Team

## Overview
This repository contains a custom static website originally created for a small consulting team. The site serves as a simple web presence, including a homepage with key information and a secondary page featuring team member professional summaries.
The website was part of a larger branding effort, implementing the team’s visual identity across web and client communication channels. Sample code for a separate email signature deliverable has also been included.

## Project Context
During implementation, platform limitations on the client’s hosting setup required the site to be recreated using the host's native site editor.
This repository represents the original front-end build, which reflects the intended design and structure prior to those constraints.

## Project Goals
- Create a clean, professional web presence for a small team
- Implement the established brand identity in a digital format
- Keep the structure lightweight and easy to navigate
- Provide clear access to contact information

## My Role
- Visual design (layout, typography, branding application)
- Front-end development (HTML, CSS, light JavaScript)

## Notes
- This repository contains a cleaned version of the original project with placeholder content to remove client-identifying information
- The live version of the site differs from this codebase due to platform constraints
- A CMS/WordPress system was explored during development.
